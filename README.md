# 布丝计算器 PWA

> 电加热元件布丝设计计算工具 · 支持矩形 / 圆形 / 梯形 · 可生成 A4 Word 报告 · 移动端 PWA

## 功能特性

| 类别 | 功能 |
|------|------|
| 形状 | 矩形 · 圆形 · 梯形（一键切换，动态显示尺寸输入） |
| 计算 | 加热面积、每平方米功率、阻值、电流、丝电流、总根数、单丝阻值、总面电阻、分组数、汇流条尺寸、布丝机频率 |
| 调整 | 手动调整功率密度 / 分组数，并反算新功率；支持功率↔电压互相反推 |
| 警告 | 丝电流 > 0.054 A、功率密度 > 900 W/m² 自动高亮提示 |
| 报告 | 一键生成 A4 Word 文档（包含输入参数、计算结果、公式过程、警告信息、调整记录） |
| 持久化 | 所有输入自动保存到浏览器本地存储，下次打开自动恢复 |
| PWA | 支持「添加到主屏幕」，安装后离线可用 |

## 支持的浏览器

- Chrome 90+ / Edge 90+（推荐，支持完整 PWA 安装）
- Safari 14+（iOS 添加到主屏幕后可离线使用）
- Firefox 88+（可使用全部功能，PWA 安装需手动）

## 使用方法

### 方式 1：本地直接打开（最快）

双击 `布丝计算器.html` 即可在默认浏览器中使用。

### 方式 2：本地 HTTP 服务器（启用 PWA 离线功能）

```bash
# 方法 A：使用 Python 3
python -m http.server 8080

# 方法 B：使用 Node.js
npx http-server -p 8080
```

然后在浏览器中访问 `http://localhost:8080/`，在菜单中选择「添加到主屏幕」即可作为 PWA 安装。

### 方式 3：部署到 GitHub Pages（推荐）

1. 把本目录全部内容推送到 GitHub 仓库的默认分支（main / master）
2. 在仓库 `Settings → Pages` 中选择分支根目录并启用
3. 几分钟后访问 `https://<你的用户名>.github.io/<仓库名>/` 即可
4. 在手机浏览器中打开，菜单里选择「添加到主屏幕」→ 获得可离线使用的原生级 App

### 移动端使用小贴士

- 📱 **Android（Chrome）**：打开页面 → 右上角菜单 → 「安装应用」或「添加到主屏幕」
- 🍎 **iOS（Safari）**：打开页面 → 分享按钮 → 「添加到主屏幕」
- ✅ 安装后点击桌面图标即可全屏、离线使用
- 📄 生成的 Word 报告在手机上可用 WPS / Microsoft Word 直接打开或分享

## 核心计算公式

1. **加热面积**（mm² → m²）
   - 矩形：长 × 宽 / 1,000,000
   - 圆形：π × (直径/2)² / 1,000,000
   - 梯形：(上底 + 下底) × 高 / 2 / 1,000,000
2. **每平方米功率** = 功率 / 面积
3. **阻值** = 电压² / 功率
4. **电流** = 电压 / 阻值
5. **电阻丝总根数** = 等效长 / 丝距 + 1
6. **单丝阻值** = 米阻 × (等效宽/1000) × 丝率
7. **总面电阻** = 单丝阻值 / 根数
8. **分组数** = 四舍五入 √(阻值 / 总面电阻)
9. **丝电流** = 电流 / (根数 / 分组数)
10. **小组汇流条长度** = 等效长 / 组数
11. **大组汇流条长度** = 小组长度 × 2（分组数 > 1 时）

## 警告阈值

- 丝电流 > **0.054 A** — 可能出现丝蕴，需调整丝距 / 分组数
- 每平方米功率 > **900 W/m²** — 建议加装温控器

## 注意事项

1. 请根据实际材料、工艺对结果进行二次校核，本计算器仅作辅助设计
2. 由于本程序通过 Service Worker 提供离线能力，首次访问必须通过 `http(s)://` 协议（非 `file://`）才能注册 SW
3. 若要删除本地存储的数据：浏览器开发者工具 → Application → Local Storage → 删除 `busi_calc_v2` 键
